## 💻 Technologies
* TypeScript
* React
* Redux Toolkit
* React Hook Form
* Framer Motion
* EmailJS
* React Scroll
* SCSS

## 🚀 Features

* Scroll-linked animation
* Adaptive layout
* Validated form for sending emails to my mailbox

## 🤖 How to run

```bash
yarn install
#
yarn start
```

Go to [http://localhost:3000](http://localhost:3000) to see result
